# Scaffold_Bancolombia_CRUD

## Información del Proyecto
- **Descripción:** Este proyecto es un CRUD (Crear, Leer, Actualizar, Eliminar) de usuarios, desarrollado utilizando el **Scaffold Clean Architecture** de Bancolombia. El objetivo es demostrar cómo implementar un sistema básico de gestión de usuarios siguiendo las mejores prácticas de arquitectura limpia, ideal para entornos bancarios o empresariales
- **Creado:** 17/06/2025
- **Lenguajes:** Java, Batchfile, Dockerfile, url
